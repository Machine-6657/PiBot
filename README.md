### 树莓派MCP语音智能体（Qwen-Agent框架）
#### 1）安装千问智能体框架：pip install -U "qwen-agent[gui,rag,code_interpreter,mcp]"（gui可选）
#### 2）修改env文件里的key
#### 3）树莓派python虚拟环境下运行：python voice-qwen3-mcp.py
#### 4）可自行配置更多MCP在文件：mcp_server_config.json
